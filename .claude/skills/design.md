# Design Skill

Colors: CSS variables only. Never hardcode hex in components.
Tailwind: use for spacing/layout. Colors via var(--gold) etc.
Framer Motion: AnimatePresence for mount/unmount. whileInView for scroll.
Fonts: Cormorant Garamond for headings/display. DM Sans for body.
Animations: aurora (8-16s), typewriter (80ms/char), flip words (2.5s).
Mobile: Floating Dock replaces navbar links. Table replaced by expandable cards.
Images: Unsplash URLs with ?w=1200&q=80 params.
Noise: SVG feTurbulence opacity 0.03 max.
Gold rule: only one accent color. Rose (#C5BAC4) as whisper only.
