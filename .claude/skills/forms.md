# Forms Skill

Library: React Hook Form + Zod validation.
Schema in /lib/validations.ts.
True Focus: animate gold border on active input via framer motion.
Errors: inline below field, slide-in animation, red border.
Steps: manage via useState currentStep, validate before proceeding.
Summary: derive from form state, update on every change.
Phone: validate Malaysian format (+60, min 10 digits).
Submit: server action, show loading state, handle errors.
Calendar: custom built, no library. Block past + booked dates.
